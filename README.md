Objectives

Understand customer demographics and purchasing patterns
Identify high-value customer segments
Analyze top-selling product categories
Examine state-wise and occupation-wise sales trends

🛠 Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📊 Project Steps
Data Cleaning
Removed null values and duplicates
Dropped irrelevant columns
Ensured correct data types

Data Exploration
Analyzed customer distribution by gender, age group, and marital status
Explored purchasing patterns across states and occupations
Evaluated product category performance

Data Visualization
Created charts and graphs to visualize sales distribution and trends
Compared purchasing behavior across different customer segments

🔍 Key Insights
Women customers aged 26–35 contributed the highest number of purchases.
Customers from IT, Healthcare, and Aviation sectors showed strong purchasing activity.
Clothing, Food, and Electronics emerged as the top-performing product categories.
Uttar Pradesh, Maharashtra, and Karnataka generated the highest sales.

📈 Business Impact
The analysis helps businesses:
Identify target customer groups
Optimize marketing campaigns during festive seasons
Focus on high-demand product categories to maximize revenue.
